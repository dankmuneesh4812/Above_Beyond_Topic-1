IoT Applications

Activity 1: IoT Apps

IoT Applications are sevices or softwares that are connected with the internet that provide real-time data through connected devices. They work without human intervention and provide accurate data depending upon the environmental factors. The IoT applications are useful in Home automation, environment monitering and supply chain management system.

The main features of Iot Applications are:

  • Real Time connectivity
  • Data Analyzation
  • Decision making
  • Safety

Activity 2: Smart Buildings

Smart Buildings are the self-capable building that can monitor conditions and take decisions. They are smart enough to tackle any problem and provide solutions to the existing problems without much human involvement. Their maintenance should be low cost and are energy efficients. Their approach should be eco-friendly and implement many technologies such as rain water harvesting, solar panel integrated grid system etc. The main features of Smart Buildings are:

  • Real Time connectivity
  • High Security and Reliability
  • Self-sustainable
  • Minimal error in false case scenario
  • Low-cost maintenance

Activity 3: Smart Buildings

Control Interace are the dashboard that can be used by the users to monitor or view different parameters. They should be designed in more user-friendly way that provide security to the backup data as well. The Control Interface are useful in many scenarios like Home automation, weather monitering and supply chain management system. In home automation, the control interface can be made for registered and non-registered user and different features can be provided and the different features can be provided such as entry notification to the registred users. For unregistered users, an alert system can be activitated after failing to recognise the face.

The parts needed in making Control Interface are:

  • Stable Internet Connection
  • Sensors for collecting data
  • GUI dashboard
  • Micro-processor/controller boards like Arduino, Raspberry pi etc.
